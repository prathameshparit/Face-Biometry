Face Biometry

Facial recognition technology has been in use for decades, and remains one of the most widely used biometrics. Facial recognition technology uses the layout of facial features and their distance from one another for identification against a “gallery” of faces with similar characteristics.

Features

1. Recognises face on camera portal and tries to find and match patterns with the data it has been trained on.

2. If recognized it authorises the face and name of person is appeared on the screen.

3. If not it says unauthorized on the screen.

4. If recognized the report of the particular person is been maintained in csv format with the name, date and time. Also provides download option for downloading the data.

5. If unauthorized the image of unauthorized person is been saved at the database with it's date and time when the person has arrived.

6. The web application also provides a person to enroll itself with it's name and face and then the model is trained again with new data it has recieved.
